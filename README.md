# 🐱🐶 Cat vs Dog Classifier (Streamlit)

This is a Machine Learning web app built using Streamlit that classifies images as Cat or Dog.

## 🚀 Features
- Upload an image
- Predict Cat or Dog
- Simple UI using Streamlit

## 🧠 Model
- Algorithm: SVM (Support Vector Machine)
- Image preprocessing: Grayscale + Resize (64x64)

## ▶️ Run Locally
```bash
pip install -r requirements.txt
streamlit run app.py
